    compact: HashMap<CidV1DagCborBlake2b256, V>,
    uncompact: HashMap<Uncompactable, V>,
}
Expand description

A space-optimised hash map of [Cid]s, matching the API for std::collections::HashMap.

We accept the implementation complexity of per-compaction-method HashMaps for the space savings, which are constant per-variant, rather than constant per-item.

This is dramatic for large maps! Using, e.g frozen_vec::SmallCid will cost 25% more per-CID in the median case (32 B vs 40 B)
